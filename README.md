# cop-3701-Insurance-Claims-Risk-Analytics-Database
Project Scope: This project intends on acheiveing goals such as:
      -Creating a relational database schema.
      -Creating tables to show to each one of our users.
      -Writing SQL queries to track claim history, scan trends in payouts, and indentify any fraud-risk variables.
      -Analyse historical claims with the help of indexes for optimization.
      -Importing sample data into the database as a test.

Users: There are a variety of users in this porject including:
      -Database administrators for insurance claims.
      -Insurance analysts.
      -Fraud detection teams/groups.

Data sources: 
      -Dataset - Sample Insurance Claim Prediction Dataset
      -Hyperlink - https://www.kaggle.com/datasets/easonlai/sample-insurance-claim-prediction-dataset
      -Description - This dataset compares whether a claim occured along with its charges against the demographic and other health-related variables.
